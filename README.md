# Video Production: Class 1 - Introduction to AI Agents

This repository contains production materials for the asynchronous lecture videos in **Module 3: Agents, Orchestration and Tracing** of the AI Engineering Career program. It includes scripts, slides, example code, sample data, and recording notes for each video in **Class 1: Introduction to AI Agents**.

## Videos

- **Video 1:** Agent Reasoning and Tool Use — From LLM Pipelines to Agents
- **Video 2:** The ReAct Prompting Framework — Theory and Practice
- **Video 3:** Designing and Integrating Custom Tools in Agent Workflows
